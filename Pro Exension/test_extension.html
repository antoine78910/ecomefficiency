📋 Instructions de Test

  1. Installez l'extension en mode développeur dans Chrome
  2. Cliquez sur un lien de test ci-dessus
  3. Observez le comportement :
    • Pour Pipiads : Écran de chargement "ECOM EFFICIENCY" avec spinner

🔧 Debugging

Pour voir les logs de l'extension :

  1. Ouvrez les outils de développement (F12)
  2. Allez dans l'onglet "Console"
  3. Recherchez les logs préfixés par :
    • [PIPIADS-EN] - Pipiads version anglaise
    • [PIPIADS-FR] - Pipiads version française
    • Logs ElevenLabs sans préfixe spécifique
