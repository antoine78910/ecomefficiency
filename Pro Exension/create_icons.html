ECOM EFFICIENCY

Générateur d'icônes pour l'extension Chrome

📋 Instructions

  1. Cliquez sur "Télécharger" pour chaque taille d'icône
  2. Renommez les fichiers téléchargés :
    • 16x16 → icon16.png
    • 48x48 → icon48.png
    • 128x128 → icon128.png
  3. Placez les fichiers dans le dossier de votre extension
  4. Mettez à jour le manifest.json (voir code ci-dessous)

🔧 Code pour manifest.json

Ajoutez ce code à votre manifest.json après avoir créé les icônes :

  "icons": {
    "16": "icon16.png",
    "48": "icon48.png",
    "128": "icon128.png"
  },
  "action": {
    "default_popup": "popup.html",
    "default_icon": {
      "16": "icon16.png",
      "48": "icon48.png",
      "128": "icon128.png"
    }
  }
